## Test Commands

```bash

pytest tests/ -v

pytest tests/test_database.py tests/test_wal_concurrency.py -v

# Privacy tests
pytest tests/test_privacy.py -v

pytest tests/test_dashboard_empty.py tests/test_budget_boundary.py -v
```

## Test Fixtures

Each test file creates its own SQLite `:memory:` database and initializes schema. No shared state between tests. Factory functions:

```python
# conftest.py
def make_user(name="爸爸", auth_method="web_pin", is_admin=False)

```

## Critical Tests (must pass before deploy)

1. **test_wal_concurrency.py** — validates the entire two-process architecture
2. **test_privacy.py** — validates the core privacy model
3. **test_dashboard_empty.py** — prevents white-screen crash on new install
4. **test_budget_boundary.py** — prevents silent budget calculation errors
5. **test_parser.py::test_reject_malformed** — prevents ReDoS attacks
