# 生产上线检查清单（飞书物资管理系统）

## 1. 环境与配置
- [ ] 已创建并填写 `.env`，至少包含：`FEISHU_APP_ID`、`FEISHU_APP_SECRET`、`RBAC_ADMIN_OPEN_IDS`
- [ ] 如果要发飞书消息：`NOTIFY_ENABLE=true` 且 `FEISHU_TENANT_ACCESS_TOKEN` 已配置
- [ ] `APP_HOME_URL` 已设置为公网可访问地址（例如 `https://wzgl.xxx.com`）
- [ ] 如启用审批流：`APP_REQUIRE_BORROW_APPROVAL=true`，且管理员白名单不为空
- [ ] 数据库路径 `APP_DB_PATH` 已确认，并已纳入备份策略

## 2. 飞书开放平台
- [ ] 已配置网页应用桌面端/移动端主页地址
- [ ] 已配置 H5 可信域名
- [ ] 已配置 OAuth 回调地址（与 `FEISHU_REDIRECT_URI` 保持一致）
- [ ] 应用权限已包含用户基础信息读取能力
- [ ] 应用已发布到目标租户并可见

## 3. 业务功能验收
- [ ] 管理员可新增/上下架/调整库存并可见审计日志
- [ ] 普通用户可查看物资并提交借用
- [ ] 审批流开启时：借用单能经历 `pending_approval -> borrowed/rejected`
- [ ] 部分归还可执行，状态会变为 `partially_returned`
- [ ] 全部归还后状态为 `returned` 且库存回补正确
- [ ] 借用单详情可查看通知轨迹与审计轨迹

## 4. 通知与任务
- [ ] 到期提醒可发送（`due_soon`）
- [ ] 逾期提醒可发送（`overdue`）
- [ ] 逾期管理员抄送可发送（`overdue_admin_cc`）
- [ ] 详情页“管理员催还一次”可发送（`manual_remind`）
- [ ] 通知失败会记录错误并重试，日志可见

## 5. 安全与运维
- [ ] `.env` 已加入忽略并未提交到仓库
- [ ] 数据目录已做好定期备份（建议每日）
- [ ] 具备基础监控：进程存活、任务调度、消息失败率
- [ ] 故障回滚方案明确（可回退 `.env` 和应用版本）

## 6. 发布后冒烟测试（建议 10 分钟）
- [ ] 管理员登录成功，菜单完整
- [ ] 普通用户登录成功，无管理员菜单
- [ ] 创建借用单并审批通过
- [ ] 发送一条手动催还消息并能从卡片跳详情页
- [ ] 执行部分归还 + 全部归还，库存与状态正确
